class Thread(name)¶
Parameters

name (str) – Name for the new thread

Raises

  • ValueError – if thread with name name is exists

  • MultiThreadingError – if an exception is thrown in function in the queue

start()¶

Forces the thread to start executing functions in queue.

end()¶

Releases the thread so that the program can exit. As soon as the thread finishes performing functions in the queue, it exits (and there will be no wait). Don’t confuse with Thread.stop!

stop()¶

Immediately stops the stream.
